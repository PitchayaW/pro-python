# เรียนไพธอนแบบอินเทอร์แอกทีฟ

เว็บแอปสำหรับสอน Python เบื้องต้นแบบ interactive ใช้ Streamlit
นักศึกษาเขียนโค้ดในเบราว์เซอร์ได้จริง มีระบบตรวจคำตอบอัตโนมัติ
และบันทึกความก้าวหน้าลง Google Sheet ให้อาจารย์ตรวจ

## หัวข้อ
```
python-course/
├── บทที่ 0 — ความเป็นมาของไพธอน
├── บทที่ 1 — การติดตั้งและใช้งานไพธอน
├── บทที่ 2 — ตัวแปรและ Data Type 
├── บทที่ 3 — Data Structure 
├── บทที่ 4 — Conditions 
├── บทที่ 5 — Loops 
├── บทที่ 6 — Functions 
├── บทที่ 7 — Error Handling 
├── บทที่ 8 — ฟังก์ชันสำเร็จรูปและ Library 
├── บทที่ 9 — NumPy และ Linear Algebra เบื้องต้น 
├── บทที่ 10 — อ่านและสำรวจข้อมูล 
└── บทที่ 11 — จัดการข้อมูล 
```

## ข้อจำกัดที่ควรรู้

- Streamlit Community Cloud จำกัด RAM ที่ 1 GB ต่อแอป ถ้านักศึกษาเข้าพร้อมกันมากและรันโค้ดหนัก ๆ พร้อมกันอาจหน่วง
- แอปจะ "หลับ" ถ้าไม่มีคนใช้นานเกินไป และใช้เวลาราว 10-30 วินาทีในการ "ตื่น" ตอนมีคนเข้าใหม่
- การรันโค้ดนักศึกษาแต่ละครั้งใช้ subprocess แยก มี timeout 5 วินาทีต่อครั้ง กัน infinite loop
